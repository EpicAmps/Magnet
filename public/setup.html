Manual Setup (for advanced users):

1
Open Shortcuts app → Tap "+" to create new
2
Add "Get Text from Input" action
3
Add "Get Contents of URL" → Set Method to POST
URL: https://magnet-mu.vercel.app/api/note
4
Set JSON body with:
fridgeId:
content: [Select "Text from Input"]
sender: "iPhone"
5
Add "Show Notification" → Enable "Use with Share Sheet"
